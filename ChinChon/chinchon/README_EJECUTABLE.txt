========================================
   JUEGO CHIN-CHON - INSTRUCCIONES
========================================

ARCHIVOS EJECUTABLES:
---------------------

1. ejecutar.bat
   - Ejecuta el juego directamente desde los archivos .class compilados
   - Requiere que los archivos .class estén presentes

2. ejecutarJAR.bat
   - Ejecuta el juego desde el archivo JAR compilado (ChinChon.jar)
   - Método recomendado para distribución

3. ChinChon.jar
   - Archivo JAR ejecutable
   - Puede ejecutarse directamente con: java -jar ChinChon.jar

REQUISITOS:
-----------
- Java Runtime Environment (JRE) versión 8 o superior
- El juego ha sido compilado con Java 21

INSTRUCCIONES DE USO:
---------------------

Opción 1: Usar el script ejecutarJAR.bat (RECOMENDADO)
  1. Hacer doble clic en "ejecutarJAR.bat"
  2. El juego se iniciará automáticamente

Opción 2: Usar el script ejecutar.bat
  1. Hacer doble clic en "ejecutar.bat"
  2. El juego se iniciará automáticamente

Opción 3: Ejecutar desde línea de comandos
  1. Abrir una terminal en esta carpeta
  2. Ejecutar: java -jar ChinChon.jar
   O ejecutar: java -cp . chinchon.Main

NOTAS:
------
- El juego se ejecuta en modo consola
- Sigue las instrucciones en pantalla para jugar
- El juego está configurado para 4 jugadores con 2 comodines
